Inilah Ceritaku

Nama saya adalah Aris Firmansah. Saya berasal dari suatu desa kecil di Malang Timur. Desa yang asri dan saya sangat menyukainya.
Saya akan bercerita tentang awal masuk di sini, di Universitas Negeri Malang. Untuk bisa masuk di UM ini, saya rasa seperti menghadapi suatu peperangan. Bukaannya saya berlebihan, tapi itulah yang saya rasakan. Mengapa saya berkata demikian ? Untuk masuk ke UM ini saya menghadapi banyak langkah-langkah yang rumit. Dan jika saya bayangkan, saya tidak akan ingin untuk kembali ke waktu itu. Banyak tes yang aku lalui itulah yang membuat saya suram untuk mengenangnya.
Pada awalnya, saya mengikuti tes non bidik misi di SMA. Berharap bisa masuk dan diterima, tapi hasilnya nihil. Saya tidak diterima. Kemudian ada tes SNMPTN, dan saya mengikutinya. Di sini memang kesalahan saya. Di sini saya memang kurang persiapan. Saya sangat pesimis di SNMPTN ini. Setelah selesainya tes terakhir SNMPTN, saya pun memikirkan adanya tes SMPDS. Di sini saya pun mengikutinya. Dan Alhamdulillah, di tes terakhir ini saya diterima di PTI. Itulah, sedikit tulisan jelek saya. Saya memang bukan orang sastra yang pandai berkata-kata, namun itulah yang bisa saya tuliskan.
